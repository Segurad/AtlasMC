# AtlasMC

AtlasMC is a new Minecraft Server Software<br>
It is designed to run Multiple Serverinstances in one JVM using Multithreading<br>
!!! <u>AtlasMC is <b>NOT</b> designed to give a Vanilla experience</u> !!!
